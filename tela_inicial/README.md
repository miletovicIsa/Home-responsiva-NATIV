# 🌿 NATIV — Home Responsiva de Cosméticos Naturais

Landing page responsiva desenvolvida para representar a marca fictícia NATIV, focada em cosméticos naturais, sustentabilidade e bem-estar.

O projeto foi construído com foco em:
- responsividade
- organização modular do CSS
- semântica HTML
- experiência visual moderna

---


## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Arquitetura CSS modular

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando abordagem responsiva para diferentes tamanhos de tela:

- Mobile
- Tablet
- Desktop

---

## 🎨 Funcionalidades da interface

- Menu de navegação responsivo
- Hero section institucional
- Sessão de categorias de produtos
- Área de valores da marca
- Sessão de produtos mais vendidos
- Área "Sobre"
- Footer institucional
